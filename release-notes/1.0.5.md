# Solem Toolkit 1.0.5

## What's changed

- Fixed a packaging error in `api.py` that left an unresolved merge marker in the published integration.

## Why this release

Version `1.0.4` could fail to load in Home Assistant with a `SyntaxError` during import. This release contains the same functional changes, but with the broken merge marker removed so the integration can start normally.

## Upgrade notes

- Update to `1.0.5` in HACS.
- Restart Home Assistant after the update.
